## Ideia do Memento

O **Memento Pattern** é usado para **capturar e restaurar o estado interno de um objeto** sem violar o encapsulamento.
É muito comum em:

- **Undo/Redo** em editores de texto.

- **Histórico de estados** em sistemas.

- **Checkpoint** de jogos.
